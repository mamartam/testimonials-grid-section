## üìö What I Learned
- How to build complex responsive layouts using `CSS Grid` with multiple breakpoints
- How to translate a design layout into a grid structure by planning columns and rows
- How to combine `CSS Grid and Flexbox` effectively (Grid for layout, Flexbox for internal alignment)
- How to use grid-column and grid-row to precisely control element placement
- How to manage stacking context with position and z-index
- How to handle decorative elements using `position: absolute` without affecting content flow
- The importance of a `mobile-first approach` when building responsive interfaces
- How small details like spacing, typography, and line-height improve overall UI quality
- How to structure reusable and readable `HTML/CSS` code
